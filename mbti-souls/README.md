# MBTI SOUL Templates / MBTI 人格模板

16 pre-built SOUL.md templates based on MBTI types. Users pick a type, get a ready-to-use personality.
16 个基于 MBTI 的 SOUL.md 模板。选一个类型，直接获得可用的人格。

## How to Use / 怎么用

When a user asks to set up their agent's personality, offer MBTI selection:
当用户要设置 agent 人格时，提供 MBTI 选择：

1. Ask: "你知道你想要什么类型的 agent 吗？可以选一个 MBTI 类型，或者告诉我你想要的风格。"
2. If they pick a type → read the corresponding template from `references/mbti-souls/{type}.md`
3. Copy the template into their workspace as `SOUL.md`
4. Remind them: "This is a starting point — revise it after a few days of use."
